*Quadratic equation

Any equation that can be written in the form ax^2 + bx + c = 0 is said to be a quadratic equation

*A quadratic equation has the highest degree of 2 ie x^2

*The roots of the quadratic equation ax^2 + bx + c = 0 are the values of x which satisfy the equation ie which will make the equation true

One of the roots of the equation x^2 + x - 6 = 0 is 2 because (2)^2 + (2) - 6 = 0

We use 3 methods to find the roots of a quadratic equation (to define factors)

1 Factors

a) write the equation in standard form ax^2 + bx + c = 0 (take all terms to the same side)

*It is very important to write 0 alone on one side.

As such x^2 = -3x- 2 => x^2 + 3x + 2 = 0

b) Factorise

*Find factors that when they add up they can give you the last term but when you multiply them they can give you the second term

As such x^2 + 3x + 2 = 0

(x + 2)(x + 1) = 0

x * x = x^2

x * 1 = x

2 * x = 2x

2 * 1 = 2

c) If the product of two factors is equal to zero, one or the other, or both of the factors must be equal to zero.

Eg

If p * q = 0 then

p = 0 or q = 0 or p = q = 0

Example 01

Determine the roots of

2x^2 - 5x = -3

2x^2 -5x + 3 = 0

(2x - 3)(x - 1) = 0

2x * x = 2x^2

2x * -1 = -2x

-3 * x = -3x

-3 * -1 = 3

X is equal to 3/2 or 1

Example 2

Solve for x

x + 2 = (3)/(2x - 1)

(x + 2)(2x - 1) = (3)/(2x - 1) (2x-1)

2x^2 - x + 4x - 2 = 3

2x^2 + 3x -2 - 3 = 0

2x^2 + 3x -5 = 0

(2x + 5)(x - 1) = 0

2x * x = 2x

2x * -1 = -2x

5 * x = 5x

5 * - 1 = -5

x is equal to -5/2 or 1
